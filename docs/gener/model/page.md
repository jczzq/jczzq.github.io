| views          | [View] | -      | true     | -      | 视图列表                  |
| viewNo         | Number | -      | true     | -      | 最大视图编号              |
| dict           | [Dict] | -      | true     | -      | 字典列表                  |
| createdAt      | Date   | -      | true     | -      | 创建时间                  |
